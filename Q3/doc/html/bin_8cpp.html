Função para conversão recursiva de um número em base decimal para binário.

Parâmetros
numRepresenta o número passado pelo usuário para realizar a conversão.
potenciaNúmero para controle e representação na conversão de decimal para binário. Faz-se uma analogia dessa variável ao controle de unidades, dezenas, centenas, etc.
binarioRepresenta o resultado da conversão da base decimal para binário.
Retorna
Retorna o resultado da conversão, com o número inserido em binário.
